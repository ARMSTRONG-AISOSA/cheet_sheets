## ðŸ”¹ Option 2: Using Node.js + `live-server` (from terminal)

If you want to run Live Server **directly from the terminal**, you can install it globally.

1. Make sure you have **Node.js + npm** installed.
   Check:
   ```bash
   node -v
   npm -v
   ```

2. Install `live-server` globally:
   ```bash
   npm install -g live-server
   ```

3. Navigate to your project folder in the terminal:
   ```bash
   cd path/to/your/project
   ```

4. Run Live Server:
   ```bash
   live-server
   ```

5. It will open your default browser at something like:
   ```
   http://127.0.0.1:8080/
   ```
